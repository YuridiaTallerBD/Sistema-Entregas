Zonas como base de la fragmentación horizontal

Las zonas NORTE, CENTRO y SUR definidas aquí se usan como claves de fragmentación horizontal para distribuir los repartidores y los pedidos entre las distintas bases de datos (base_norte, base_centro, base_sur).

Si cambias o agregas zonas, estos códigos se pueden usar para crear nuevos fragmentos lógicos.
